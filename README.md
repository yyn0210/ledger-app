# 全平台智能记账 App

📸 拍一拍，记一记，财务自由更容易

## 项目结构

- `backend/` - Spring Boot 3.x 后端
- `frontend/` - Vue 3.x Web 前端
- `design/` - 设计资源

## 快速开始

### 后端
```bash
cd backend && docker-compose up -d
```

### 前端
```bash
cd frontend && npm install && npm run dev
```
