## 组件库概览

```
组件库结构
├── 基础组件 (Base Components)
│   ├── 按钮 (Buttons)
│   ├── 输入框 (Inputs)
│   ├── 选择器 (Pickers)
│   ├── 开关 (Switches)
│   ├── 复选框 (Checkboxes)
│   └── 单选框 (Radio Buttons)
│
├── 业务组件 (Business Components)
│   ├── 金额输入 (Amount Input)
│   ├── 分类选择器 (Category Picker)
│   ├── 账户选择器 (Account Picker)
│   ├── 日期选择器 (Date Picker)
│   ├── 交易卡片 (Transaction Card)
│   └── 统计图表 (Charts)
│
└── 反馈组件 (Feedback Components)
    ├── Toast 提示
    ├── Dialog 对话框
    ├── Loading 加载
    ├── 空状态 (Empty State)
    └── 错误页面 (Error Page)
```

## 四、组件状态机

### 按钮状态机
```
Normal ──[hover]──> Hover
   │                   │
[disabled]         [leave]
   │                   │
   v                   v
Disabled <─────── Normal
```

### 输入框状态机
```
Normal ──[focus]──> Focus
   │                   │
[disabled]         [blur]
   │                   │
   v                   v
Disabled <─────── Normal
   │
[error]
   │
   v
 Error
```

### Dialog 状态机
```
Hidden ──[open]──> Opening ──[动画完成]──> Open
                          │
                      [close]
                          │
                          v
                     Closing ──[动画完成]──> Hidden
```
