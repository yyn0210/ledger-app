# Figma 设计文件结构规划

## 🎬 Page 4: Prototypes

### User Flow - 用户流程

```
启动 → 登录/注册 → 首页 → 记账 → 完成
                          ↓
              拍照/语音 → AI 识别 → 确认 → 完成
                          ↓
              手动输入 → 选择分类 → 确认 → 完成
```

### Interactions - 交互动效

| 交互 | 触发 | 动画 | 时长 |
|------|------|------|------|
| 按钮点击 | Click | Scale 0.95 | 150ms |
| 页面切换 | Navigate | Slide Left | 300ms |
| 弹窗出现 | Open | Fade + Scale | 250ms |
| 记账成功 | Submit | Confetti | 1000ms |
| 下拉刷新 | Pull | Rotate | 500ms |

## 📋 Figma 组件命名规范

### 命名格式

```
Category/Component/Variant
例：Buttons/Primary/Large
例：Inputs/Text/Default
```

### Auto Layout 设置

```
Buttons:
- Padding: 12px 24px
- Gap: 8px
- Resize: Hug contents

Cards:
- Padding: 16px
- Gap: 12px
- Resize: Fill container
```

### Constraints 设置

```
Desktop:
- Width: Fill container
- Height: Hug contents

Mobile:
- Width: Fixed (375px)
- Height: Hug contents
```

## 📤 导出规范

### 切图格式

| 资源类型 | 格式 | 倍率 |
|----------|------|------|
| 图标 | SVG | 1x |
| 图片 | PNG | 1x, 2x, 3x |
| 背景 | PNG/JPG | 1x |

### 导出设置

```
Icons:
- Format: SVG
- Suffix: @1x, @2x, @3x

Images:
- Format: PNG
- Suffix: @1x, @2x, @3x
```
