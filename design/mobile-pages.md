### 交互流程

**1. 正常记账流程**
```
1. 点击底部 Tab「记账」或首页「记账」按钮
   ↓
2. 底部弹出记账面板（从下向上滑入，300ms）
   ↓
3. 自动弹出数字键盘
   ↓
4. 输入金额（数字滚动显示）
   ↓
5. 选择分类（点击分类图标）
   ↓
6. 选择账户（可选，默认上次使用）
   ↓
7. 选择日期（可选，默认今天）
   ↓
8. 添加备注（可选）
   ↓
9. 点击「保存」按钮
   ↓
10. 显示成功动画（600ms）
    - ✓ 图标放大 + 粒子效果
    - 金额数字滚动
    - 轻微震动反馈（Haptic）
    ↓
11. 面板关闭，返回首页
    ↓
12. Toast 提示「记账成功」
```

**2. 拍照记账流程**
```
1. 点击「拍照」按钮
   ↓
2. 打开相机界面
   ↓
3. 拍照或选择相册图片
   ↓
4. 显示识别中动画（2-5 秒）
   - AI 思考效果
   - 进度条
   ↓
5. 显示识别结果确认页
   - 商户名（可编辑）
   - 金额（可编辑）
   - 日期（可编辑）
   - 推荐分类（可修改）
   ↓
6. 用户确认/修改
   ↓
7. 保存，显示成功动画
```

**3. 语音记账流程**
```
1. 点击「语音」按钮
   ↓
2. 显示语音输入界面
   - 麦克风图标
   - 声波动画
   - 提示「请说话...」
   ↓
3. 用户说话（如「今天打车花了 35 块」）
   ↓
4. 显示识别中（1-2 秒）
   ↓
5. 显示识别结果
   - 文字：「今天打车花了 35 块」
   - 提取信息：
     - 金额：¥35
     - 分类：交通出行
     - 日期：今天
   ↓
6. 用户确认/修改
   ↓
7. 保存，显示成功动画
```

### 成功动画详细设计

```
帧 1 (0ms)
┌─────────────────┐
│                 │
│       ○         │  ✓ 图标出现（缩放 0）
│                 │
└─────────────────┘

帧 2 (200ms)
┌─────────────────┐
│                 │
│       ✓         │  ✓ 图标放大到 1.2 倍
│                 │
└─────────────────┘

帧 3 (400ms)
┌─────────────────┐
│    ✨    ✨     │
│       ✓         │  粒子效果扩散
│    ✨    ✨     │  ✓ 图标回弹到 1.0 倍
└─────────────────┘

帧 4 (600ms)
┌─────────────────┐
│                 │
```

**动画参数**
- 总时长：600ms
- 缓动函数：cubic-bezier(0.68, -0.55, 0.265, 1.55) (弹性)
- 震动反馈：成功时触发（iOS Haptic Feedback）
- 音效：可选「叮」声（清脆）

## 深色模式

### 颜色映射

```
浅色模式 → 深色模式
#ffffff → #1a1a1a (背景)
#f7f8fa → #252525 (次要背景)
#1a1a1a → #ffffff (文字)
#4a4a4a → #e0e0e0 (次要文字)
#e8ecef → #404040 (边框)
```

### 切换方式

**系统跟随**
- 默认跟随系统深色/浅色模式
- 使用系统 API 监听主题变化

**手动切换**
- 设置页提供开关
- 切换时渐变动画（300ms）

### 深色模式注意事项

- 避免纯黑背景（使用#1a1a1a）
- 降低饱和度（减少视觉疲劳）
- 保持对比度（WCAG AA 标准）
- 测试所有页面状态
