### 3.3 推荐逻辑

**特征权重**
```

- 关键词匹配（超市→购物）
- 历史习惯（用户常分类别）

金额范围：25%
- 小额（<50）：餐饮/交通
- 中额（50-500）：购物/餐饮
- 大额（>500）：房租/电器

消费时间：15%
- 早餐时间（7-9 点）：餐饮
- 午餐时间（11-14 点）：餐饮
- 晚餐时间（17-20 点）：餐饮
- 周末：娱乐/购物概率增加

消费地点：10%
- 商圈：购物/餐饮
- 住宅区：居家/超市
- 办公区：餐饮/交通

历史习惯：10%
- 用户对该商户的历史分类
- 类似商户的分类习惯
```

## 6. 数据埋点

### 6.1 关键事件

**拍照记账**
```
ocr_start: 开始拍照
ocr_capture: 完成拍照
ocr_upload: 图片上传成功
ocr_result: 识别成功（含字段置信度）
ocr_confirm: 用户确认记账
ocr_modify: 用户修改识别结果
ocr_fail: 识别失败
ocr_retry: 用户重试
ocr_manual: 用户放弃，转手动记账
```

**语音记账**
```
voice_start: 开始录音
voice_end: 录音结束
voice_upload: 音频上传成功
voice_result: 识别成功（含置信度）
voice_confirm: 用户确认记账
voice_modify: 用户修改识别结果
voice_fail: 识别失败
voice_retry: 用户重试
```

**AI 分类**
```
ai_recommend: AI 推荐分类（含推荐列表）
ai_accept: 用户接受推荐
ai_reject: 用户拒绝推荐
ai_modify: 用户修改分类
ai_learn: 学习用户行为（含学习结果）
```

### 6.2 性能指标

**识别性能**
- 识别时长：P50 / P90 / P99
- 识别成功率：成功次数 / 总次数
- 识别准确率：用户确认次数 / 识别成功次数
- 平均修改率：修改字段数 / 总字段数

**用户体验**
- 功能使用率：使用智能功能用户 / 总用户
- 功能留存率：次日/7 日/30 日留存
- 用户满意度：评分/反馈
- 平均记账时长：从进入到完成
